# STARTER-lab07

## How to use the visualization tool

1. Run `make` to build the project.
2. Run `./visualizer` to generate the trace files.
3. Run `python3 -m http.server 8080 --directory ./web-viz` to serve the web-viz folder.
4. Open `http://localhost:8080` in your browser to view the visualization.
5. Switch between the `evaluation` and `training` mode using the toggle button on top.
6. Click `Previous` to go to the previous step.
7. Click `Next` to go to the next step.
8. Click `Reset` to reset the visualization to the initial state.
